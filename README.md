# 🚀 나만의 프롬프트 관리 프로그램 (Prompt Manager)

GenAI 기초부터 자동화까지 학습하며 생성된 다양한 프롬프트들을 체계적으로 분류하고 관리하는 Python 콘솔 기반 소프트웨어입니다.

## 🛠️ 개발 환경
- **Language:** Python 3.12.x
- **IDE:** VSCode
- **Version Control:** Git / GitHub

## ✨ 주요 기능 목록
1. **프롬프트 추가:** 제목, 내용, 카테고리를 입력받아 새로운 프롬프트 등록 (빈 값 예외 처리)
2. **프롬프트 목록 보기:** 등록된 모든 프롬프트를 번호, 카테고리, 즐겨찾기 상태(⭐)와 함께 출력
3. **카테고리별 조회:** 지정된 카테고리에 속한 프롬프트만 필터링하여 조회
4. **프롬프트 검색:** 키워드를 통해 제목 및 내용 검색
5. **프롬프트 상세 보기:** 선택한 번호의 프롬프트 전체 내용 포매팅 출력
6. **즐겨찾기 관리:** 특정 프롬프트의 즐겨찾기를 추가하거나 해제하는 토글 기능
7. **즐겨찾기 목록 보기:** 즐겨찾기가 등록된 프롬프트만 모아보기

## 📂 프롬프트 카테고리 종류
- 텍스트 생성 / 이미지 생성 / 영상 생성 / 페르소나 / 자동화 / 기타

## 💻 실행 방법
터미널에서 프로젝트 폴더로 이동한 후 아래 명령어를 입력합니다.
```bash
python main.py
